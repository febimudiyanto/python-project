# Copy & Paste to help you manage in different environment


## How to setup
1. install flask
```
pip install flask
```
2. clone the repo
```
git clone <this repo>
```
or setup the file

├── app.py
└── templates
    └── index.html


3. run the web app
```
python3 app.py
```

## Access the web page

http://{ip address}:5000/
